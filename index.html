WE Quota Checker
Automatically check your WE Internet quota with ease

About WE Quota Checker

WE Quota Checker is an automated tool that helps Egyptian WE ISP customers check their remaining internet quota without the hassle of manually logging in to the website each time.

This tool uses Selenium to automatically log in to your WE account and fetch your remaining quota information, displaying it in a clean, easy-to-read format.

Quick Version

Edit the script with your credentials once, then run it anytime to quickly check your quota.

Regular Version

Enter your credentials each time for added security and flexibility.

Command Line

Simple command-line interface with colorful output for easy readability.

How It Works

WE Quota Checker uses Selenium WebDriver with Firefox to:

  1. Navigate to the WE login page
  2. Input your credentials
  3. Select the "Internet" service
  4. Log in to your account
  5. Extract and display your remaining quota

The tool runs in headless mode, meaning no browser window is shown during the process.

Installation Guide

Follow these steps to get started with WE Quota Checker:

Prerequisites

  • Firefox browser installed
  • Python 3.6 or higher
  • Selenium and Colorama Python libraries

Install Dependencies

pip install -r requirements.txt

Download the Project

Clone this repository or download from GitHub.

Alternatively, download the packaged version from Mediafire.

Usage

Quick Version: Edit the checkQuotaQuick.py file and replace the username and password placeholders with your credentials. Then run:

python checkQuotaQuick.py

Regular Version: Simply run the script and enter your credentials when prompted:

python checkQuotaRegular.py
